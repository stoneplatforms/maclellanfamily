# 8.3 Suggested order of operations

1. Create S3 bucket (console or Terraform).
2. Create MediaConvert IAM role (S3 read/write, trust `mediaconvert.amazonaws.com`).
3. Get MediaConvert endpoint from AWS console.
4. Build Lambda so `infra/terraform/build/maclellanfamily-sqs-consumer.zip` exists.
5. Update Terraform to add `MEDIACONVERT_ENDPOINT`, `MEDIACONVERT_ROLE_ARN`, `AWS_REGION` to Lambda env and `iam:PassRole` for MediaConvert role.
6. Create `terraform.tfvars` with all required variables.
7. Run `terraform init` and `terraform apply`.
8. Set `SQS_QUEUE_URL` in Vercel.
