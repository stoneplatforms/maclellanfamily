# 8.2 Gaps to fix before deploy

1. **S3 bucket** – Terraform uses `var.bucket_name` but does not create it. Create manually or add Terraform resource.
2. **MediaConvert env vars** – Lambda expects `MEDIACONVERT_ENDPOINT` and `MEDIACONVERT_ROLE_ARN`; Terraform does not pass them. Video processing will fail without these.
3. **IAM PassRole for MediaConvert** – Lambda needs `iam:PassRole` on the MediaConvert role. Terraform IAM policy does not include it.
4. **`AWS_REGION` in Lambda** – Not set in Terraform env; add for SDK consistency.
5. **`build/` directory** – Terraform expects `infra/terraform/build/maclellanfamily-sqs-consumer.zip`. Run Lambda build before `terraform apply`.
6. **MediaConvert role** – Terraform does not create it. Create manually (or add to Terraform) with S3 read/write and trust for `mediaconvert.amazonaws.com`.
